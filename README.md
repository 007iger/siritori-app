# Deno しりとりアプリ

## 概要

Denoを用いて開発したWebアプリケーションです。

サーバーサイドとフロントエンドをDenoで実装し、GitHubとDeno Deployを利用してデプロイを行いました。

## 実装した機能

### 基本機能

- 直前の単語を表示
- 任意の単語を入力
- 前の単語の最後の文字と入力した単語の最初の文字を比較
- しりとりが成立した場合のみ単語を更新
- 不成立の場合はエラーメッセージを表示

### ゲーム機能

- 「ん」で終わる単語が入力されたらゲーム終了
- 過去に使用した単語が入力されたらゲーム終了
- ゲーム中・ゲーム終了後のリセット機能

### 追加機能

- 最初の単語がランダムで決定される
- 使用した単語の履歴を画面に表示

## デザインについて

見やすさを重視し、シンプルなデザインにしました。

- 水色を基調とした背景
- 現在の単語を大きく表示
- 入力欄とボタンを中央に配置
- エラーメッセージは赤色で表示
- 使用済み単語を一覧表示できるようにした

---

## 工夫した点

ゲームの状態をサーバー側で管理し、画面はAPIから取得した情報を表示する構成にしました。

また、単語の履歴を配列で管理することで、

- 同じ単語を使用したか
- 現在の単語
- 単語履歴

をまとめて管理できるようにしました。

さらに、リセット時には最初の単語がランダムに決まるようにして、毎回違うしりとりを楽しめるようにしました。
